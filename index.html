Tutoring Request Manager

A Discord bot that manages tutoring requests for Centennial High School's Math Honor Society.

Skills used: Amazon S3, Webhooks, Python

This project is a Discord bot that uses Discord.py, a Python wrapper for the discord API. Requests for tutoring are typically created with a Google Form, which triggers a webhook on submission. To access the data regarding existing requests, the bot connects to an Amazon S3 Bucket, downloads the stored CSV, and uploads the edited CSV back to the bucket.

Facebook Posting Bot

A bot that takes content from Reddit and posts it on various Facebook pages, hosted on an Amazon EC2 Instance.

Skills used: Amazon EC2, Facebook Graph API, Reddit API, Python

This program is designed to take the hottest posts from multiple subreddits from Reddit, and post them on to various Facebook pages. First, the program uses the Reddit API to get the popular posts, and then does the necessary processing on the downloaded content. Then, it uses Facebook's Graph API to post the content to a Facebook Page. This program is hosted on an Amazon EC2 Linux instance.
